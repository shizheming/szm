Length Units

em：相对于字体大小

ex：相当于字体的一半

ex

ch：数字0的宽度

rem：相对于根元素(即html元素)font-size计算值的倍数

cm：厘米

mm：毫米

in：英寸

pt：点

pc：派卡

px：像素

1in=2.54cm=25.4mm=72pt=6pc=96px

Angle Units

deg：度

grad：梯度，一个圆共400梯度

rad：弧度，一个圆共2π弧度

turn：转、圈（Turns），一个圆共1圈

90deg=100grad=0.25turn≈1.570796326794897rad

Time Units

s：秒

ms：毫秒

1s=1000ms

Functional Notations Type Values

calc()：宽度计算

width:calc(100% - 50px);margin: calc(1rem - 2px) calc(1rem - 1px);background-position: calc(50% + 20px) calc(50% + 20px), 50% 50%;只要是长度都能用这个算

attr()：属性
