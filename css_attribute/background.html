简写属性background

background:background-image background-position/background-size background-repeat background-attachment background-origin background-clip background-color

如果background-orgin和background-clip只有一个值，那么2个属性都是这个值，如果是2个属性，那么第一个是background-orgin，第二个是background-clip。

这里5张图片，但位置只写了3个background-position:10px 10px,center center,100px 100px;他需要重复2个来填补空缺，他是这样子的，写的几个算一组，然后自动重复填补空缺所以这里是这样的background-position:10px 10px,center center,100px 100px,10px 10px,center center,100px 100px;而最后一个100px 100px是没有的因为这有5张图片。

background-color

初始值：transparent

应用于：所有元素

继承：无

动画：有

background-color通过background-clip来显示区域。

background-image

取值：url()[,url()]

初始值：none

应用于：所有元素

继承：无

动画：无

使用背景图的同时最好在指定一个背景色，提高用户体验，不过这对jpg可以用，对png，gif就不行了。

多重背景，写在最前面的图片显示在最上面，并且背景颜色只能写一次同时是写在最后一张背景上。

background-repeat

取值：repeat-style repeat-style[,repeat-style repeat-style]

初始值：repeat

应用于：所有元素

继承：无

动画：无

repeat-x,repeat-y,repeat,no-repeat,space,round

background-attachment

取值：attachment[,attachment]

初始值：scroll

应用于：所有元素

继承：无

动画：无

scroll,fixed,local

当背景图像设置为scroll时，背景不跟随这个元素的滚动，但会随元素的祖先元素或窗体一起滚动。

当背景图像设置为local时，背景随这个元素的滚动，也会随元素的祖先元素或窗体一起滚动。

当背景图像设置为fixed时，背景图放置的位置就由body来确定，而不是有包含这个背景图的元素来确定位置，不过背景只在包含元素中显示。

background-position

取值：position position position position[,position position position position]

初始值：0 0

应用于：所有元素

继承：无

动画：有

left,right,top,bottom,center

我觉得设置1个值或2个值还是3个或4个值都差不多，无非3个4个值好控制点，比如一个高为1000px的元素，假设背景垂直开始的地方时980px，设2个值的时候看上去会比较大，而4个值你只要写bottom 20px就搞定了。

图片背景定位属性第一个是水平方向，所以只有left和right，第二个是垂直方向，所以只有top和bottom

图片背景定位如果只出现一个关键字，那么另外一个自动会是center，如果出现的是数值那么算是水平方向的，垂直方向自动center。

图片背景定位应用百分比，是相对于元素和原图的相应点来确定的

从元素水平方向的50%处开始放置图片水平方向的50%处，从元素垂直方向的50%处开始放置图片垂直方向的50%处。

关键字，百分比，长度值可以混用，但要注意顺序

background-clip

取值：box[,box]

初始值：border-box

应用于：所有元素

继承：无

动画：无

border-box,padding-box,content-box

background-color默认延伸至border里面，而background-image默认padding-box开始。

background-origin

取值：box[,box]

初始值：padding-box

应用于：所有元素

继承：无

动画：无

border-box,padding-box,content-box

这是属性是用于背景图片的，所得上面的背景是从padding-box还是的就是这是属性的初始值。

这里可以看到，设置border-box以后背景图就从边框开始显示了，他的background-position也跟着变化了，是根据background-origin来的。有点蛋疼的是背景色会跟着最后一张图来。最后再加上background-clip的效果，切的第二张图是content-box并不会影响到第一张图，他content-box意外的地方还是看得见的，也就是说不管几张图各自的属性只影响自己，背景色是属于最后一张的。

background-size

取值：bg-size[,bg-size]

初始值：auto

应用于：所有元素

继承：无

动画：无

length,auto,cover,contain

如果填数字，第一个是宽度，第二个是高度，如果只有一个默认作为宽度，高度自动为auto。auto的时候是根据图片设的宽度等比例缩放的。

当background-size和round遇到一起的时候，size输入的值只有达到一定的round的程度的时候才会有效果。

background:url(img5.jpg) #ccc;height:300px;background-size:0;size设0，背景色看不出来，加上background-repeat设repeat，背景色还是看不出来，一定要是repeat意外的任何值才能看出背景色来。

-moz-radial-gradient(#58ff00 10%,rgba(222, 255, 0, 0) 20%),-moz-radial-gradient(20% 30%,circle,#58ff00, rgba(222, 255, 0, 0) 50px),-moz-linear-gradient(70deg,#f00,#ff0 15%,#080 30%)，radial径向渐变，linear线性渐变，第一组参数是旋转的角度和起始的位置，径向渐变的角度好像有点奇怪，径向渐变中写了circle就变成正圆了，不写的话是按照元素的形状来变成椭圆的，然后是颜色后面可以跟个位置的数值，不写的话他会平均分配以铺满整个元素，设置颜色的透明度后就能看见许多渐变了。

起始点直接写关键字left就好了，不用费力的让他旋转90度。

当渐变的开始位置和结束位置设成一样的时候就能产生硬扁的效果，配合background-size或是background-position或是background-repeat来使用，当然这里我就用了size，他有2个参数，第一个用于定义背景图像的宽度，第二个用于定义背景图像的高度，如果只提供一个，该值将用于定义背景图像的宽度，第2个值默认为auto，即高度为auto，此时背景图以提供的宽度作为参照来进行等比缩放。

利用2次渐变外加偏移来拼凑成的棋盘格。
