min-width,min-height,max-width,max-height：无继承

应用于除了行内非替换元素和表元素以外的所有元素

百分比：min-width相对于包含块的宽度，min-height相对于包含块的高度，max-width相对于包含块的宽度，max-height相对于包含块的高度

正常文档流中块级元素的高度有包裹性，而宽度有延伸性，就像inline-block和overflow来请浮动一样

 

加上定位的最大最小宽高的情况

定位自适应宽度与最小宽赌和高度高，取大的那个值。

自适应宽度与最大宽度和高度，取小的那个值。

最小宽高取她与高度大的那个，最大宽高取她与高度小的那个

width:无继承

应用于块级元素和替换元素

百分比：相对于包含快的宽度计算

height:无继承

应用于块级元素和替换元素

百分比：相对于包含快的高度计算

高度的百分比是根据包含快高度来的，如果包含块高度自适应，百分比就无效。

包含块没有垂直内边距和上下边框的情况下，其默认高度是从最高块级子元素的上边框到最低块级子元素的下边框的距离，所以块级子元素的垂直外边距会超出包含块，不过如果包含块有垂直内边距和上下边框的话，其高度就从最高块级子元素的上外边距到最低子元素的下外边距。

没有上下边框上下内边距的块

没有上下边框，有上下内边距的块

有上下边框，没有上下内边距的块

包含块设置了高度后，后代元素的内外边距即使超过包含块也不会影响文档流
