list-style简写属性

list-style:list-style-type,list-style-position,list-style-image

3个值可以任意顺序，而且都能忽略

应用于dispaly值为list-item的元素

初始值：相对应各个属性

list-style-image

初始值：none

应用于：list-item

继承：有

动画：无

有图片会代替list-style-type的标记，没图片会是默认的list-style-type标记。

  • list-style-image有继承
    • 里面的也会继承父元素的图片

这个属性是作用于list-item的元素上的，这里我写在h1上了，h1不是list-item元素是block元素，但还是拥有相同的效果，因为它是由继承的，就算再套个span写在span上页是一样的，不过这个只有list-style-image有效，其他2个无效，但是为什么他们还能写在ul或ol上，这2个不是block嘛。

初始值：outside

应用于：list-item

继承：有

动画：无
